# RRA Brand Kit — README

## What's in this kit

```
rr-brand-kit/
├── BRAND_GUIDE.md        ← Complete brand rules (colours, fonts, logo usage, templates)
├── BRAND_PROMPT.md       ← System prompt for Claude (paste into Projects for auto-compliance)
├── README.md             ← You are here
├── logos/                ← 27 named logo files (PNG)
│   ├── rr-main-logo-pink.png
│   ├── rra-logo-pink-on-white.png
│   ├── rra-logo-blue-on-white.png
│   ├── rra-logo-white-on-black.png
│   ├── rra-australia-light.png
│   ├── rra-melbourne-light.png / dark.png
│   ├── lion-brand-element-*.png
│   └── ... (all location variants)
├── assets/               ← 85 raw extracted images (backgrounds, photos, templates)
├── page-renders/         ← Full-page PNG renders of every page (19 pages)
└── reference/            ← Original PDF preserved
```

## Quick Start

1. **Read** `BRAND_GUIDE.md` for all the rules
2. **Paste** the system prompt from `BRAND_PROMPT.md` into your Claude Project
3. **Upload** the `logos/` folder into Project Knowledge
4. **Build** — everything you create will be brand-compliant
